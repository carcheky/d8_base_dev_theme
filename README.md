# d8_base_dev_theme

1. Download repo
2. Rename 'themerepo' folder and move to destination
3. Rename 'themerepo.info.yml' file (you can customize this file)
4. Rename 'themerepo.libraries.yml' file
5. Rename 'themerepo.breakpoints.yml' file
6. Rename 'themerepo.layouts.yml' file
7. Rename 'themerepo.theme' file
8. Edit 'themerepo.theme' file and replace "themerepo" to your theme name
9. Search & replace "themerepo" to "yourthemename"
10. Install && enable
11. Open [Prepros](https://prepros.io) and style! (or 'install live sass compiler' in vscode, you )
